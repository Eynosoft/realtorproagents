User Registration - DON'T PROMPT IMMEDIATELY!!

It's important to let the user get engaged with the searching before prompting for Registration. You don't want to prompt right away. This has been studied and surveyed that a user will leave the website right away if immediately prompted. Our IDX allows you to set when you want to prompt for registration. You can set it to after 'x' amount of searches where you define the 'x' or after 'x' amount of detailed property views. This allows the user to get engaged first so they feel comfortable to register and they know what they're registering for.
